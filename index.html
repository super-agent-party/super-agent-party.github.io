VRM Desktop Companion

Supports importing custom VRM models. Break the 4th wall and let your exclusive character live on your desktop with rich actions and expressions, creating a unique digital companion.

  • Custom Model Import
  • 24/7 Desktop Companionship
02

Omnipotent Task Center

Give AI true agency. Agents can execute advanced tasks in the background, automatically controlling your PC to handle tedious work, from file organization to complex workflow automation.

  • System-Level Access
  • Automated Workflows
03

Multi-character Chat

Supports importing SillyTavern character cards equipped with long-term memory. You can create multiple AI characters, chat with them simultaneously in the same room, and watch them interact with each other.

04

IM Bot Matrix

Deploy your agents to mainstream social platforms with one click. Stay connected anywhere, whether for work (Feishu/DingTalk/Slack) or social networking (QQ/Telegram/Discord).

05

Virtual Streamer System

Transform into an AI VTuber. Connect seamlessly to Bilibili, YouTube, and Twitch. The agent can automatically read live chat and interact via voice, enabling 24/7 unassisted streaming.

06

AI Browser

Give agents the ability to "see" the web. With an independent browser environment, AI can automatically surf the internet to gather information, read news, and operate web apps, becoming your ultimate internet assistant.

07

Infinite Ecosystem

A powerful plugin system supporting one-click installations or custom development. From playing visual novels to smart home control, the boundaries of its features are determined entirely by your imagination.

Developer Friendly

Extremely open architecture. Provides standard OpenAI API interfaces and MCP protocol support. Easily connect SAP's agent capabilities to other applications or interact deeply via code.
