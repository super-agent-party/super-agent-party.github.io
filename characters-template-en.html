About Character Cards

These character cards use the SillyTavern V3 (chara_card_v3) format. You can download the JSON file and import it directly into Super Agent Party or any compatible AI chat application.

// chara_card_v3 format
{
  "spec": "chara_card_v3",
  "spec_version": "3.0",
  "name": "Character Name",
  "description": "Character description",
  "avatar": "url or empty",
  "personality": "...",
  "first_mes": "First greeting message",
  "mes_example": "Example dialogues",
  "system_prompt": "System prompt",
  "alternate_greetings": [...],
  "character_book": { ... }
}

Characters folder: characters/ and characters-zh/
