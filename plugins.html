Adding Your Own Plugin

Want to add your plugin to this list? You can easily update the plugins by modifying the plugins configuration file. 

// Create or modify plugins.json in the root directory
[
  {
    "name": "Your Plugin Name",
    "description": "A short description of what your plugin does",
    "author": "Your Name",
    "version": "1.0.0",
    "category": "Utility",
    "repository": "https://github.com/your-username/your-plugin"
  }
]

For more detailed information about plugin development and submission, please refer to the Super Agent Party documentation.
