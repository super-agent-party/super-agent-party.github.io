关于角色卡

这些角色卡使用 SillyTavern V3 (chara_card_v3) 格式。您可以下载 JSON 文件并直接导入到 Super Agent Party 或任何兼容的 AI 聊天应用中。

// chara_card_v3 格式
{
  "spec": "chara_card_v3",
  "spec_version": "3.0",
  "name": "角色名称",
  "description": "角色描述",
  "avatar": "图片URL或空",
  "personality": "性格描述",
  "first_mes": "首次问候语",
  "mes_example": "对话示例",
  "system_prompt": "系统提示词",
  "alternate_greetings": [...],
  "character_book": { ... }
}

角色卡文件夹： characters/ 和 characters-zh/
